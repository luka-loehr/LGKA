# LGKA Screenshots

This directory contains the localized screenshots for the LGKA app, organized for app store submissions.

## Directory Structure

```text
screenshots/
├── de/                 # German localization
│   ├── ios/
│   │   ├── phone/      # iPhone 6.5" (1290x2796)
│   │   └── tablet/     # iPad 13" (2048x2732)
│   └── android/
│       ├── phone/      # Android Phone (1080x2400)
│       └── tablet/     # Android 10" Tablet (1200x1920)
└── en/                 # English localization
    ├── ios/
    │   ├── phone/
    │   └── tablet/
    └── android/
        ├── phone/
        └── tablet/
```

## Standard Screenshot Order

1. `01_welcome.png`
2. `02_info.png`
3. `03_auth.png`
4. `04_substitution.png`
5. `05_weather.png`
6. `06_schedule.png`
7. `07_settings.png`

## Requirements

### iOS
- **Phone (6.5"):** 1290 x 2796 pixels
- **Tablet (13"):** 2048 x 2732 pixels

### Android
- **Phone:** 1080 x 2400 pixels (preferred)
- **Tablet (10"):** 1200 x 1920 pixels (preferred)
