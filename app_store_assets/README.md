# App Store Assets

This directory contains all assets needed for app store submissions, including localized screenshots and promotional banners.

## Directory Structure

- `screenshots/`: Localized app screenshots (see `screenshots/README.md`)
- `banners/`: Promotional graphics and feature graphics

## Localized Screenshots

We maintain screenshots for both German (`de`) and English (`en`) languages across iOS and Android platforms.

For detailed information on dimensions and organization, please refer to [screenshots/README.md](screenshots/README.md).
